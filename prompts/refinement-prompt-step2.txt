Design and implement a browser-based slot machine game using vanilla HTML, CSS, and JavaScript that avoids the use of external frameworks. The project should feature a clean, responsive UI with smooth CSS-driven animations for reel spins and win celebrations, including a dark/light mode toggle and accessible keyboard controls. You must architect the application by strictly following SRP and OCP principles, separating concerns into dedicated modules for game logic, state management, UI rendering, and audio.

Incorporate core gameplay mechanics such as a protected betting system, an RNG-based payout engine with a detailed symbol table, and advanced features like configurable autoplay, win streak tracking, and rare jackpot events. The game logic should include a dynamic difficulty system that adjusts payout rates based on player performance while maintaining a clear display of real-time statistics, including RTP, total spins, and a detailed history log. Ensure the code is well-structured and documented for future extensibility, utilizing reusable components and avoiding global state to maintain high-quality software standards.
