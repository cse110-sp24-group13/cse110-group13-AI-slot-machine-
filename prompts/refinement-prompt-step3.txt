Create a polished, browser-based parody of AI inference as a slot machine using only HTML, CSS, and JavaScript. Temperature should not be cosmetic; it should noticeably change volatility, making higher settings produce riskier, less stable results. Include a live system console that turns every result into a funny but readable AI status report.
Make the app feel alive with smooth reel motion, satisfying button states, strong result screens, and responsive layout. Use satire to reinforce the mechanic, not confuse it: every label, symbol, and payout should connect to an AI idea in a way users can quickly understand.
Keep the interface accessible, the visuals cohesive, and the code easy to maintain with clean separation between structure, styling, and logic. Prioritize clarity, delight, and replayability over feature bloat.
