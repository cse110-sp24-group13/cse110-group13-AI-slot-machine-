## App Quality
- Does it actually run in the browser without errors?
- Do the reels spin correctly — is the core mechanic functional?
- Is the AI/token theming present and recognizable?
- Is there a visible win/loss condition?
- Is there a token balance that updates correctly?
- Does it feel like a complete experience or does it feel half-finished?
- Does it look intentional or like a raw skeleton?

## Code Quality
- How long is it? Does the AI over-engineer or under-build?
- Is it actually vanilla HTML/CSS/JS?
- Is it self-contained? Does it run by just opening the HTML file?
- Is the code readable? Do the variable names make sense, logical structure?
- Are there obvious bugs even if the app appears to work?
- Is game state managed cleanly?
- Are there any security red flags?
